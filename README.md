进入服务器

项目在 /root/yesskins/front-end/中
使用 serve 启动前端项目
在 dist 目录下 使用 
```
nohup serve -p 7777 &
``` 
指定端口号启动项目
不加 & 在 ctrl+c 后就会结束
加了之后会运行在服务器后台
